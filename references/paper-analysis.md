# 论文分析与配图规划框架

## 第一步：论文结构分析

### 1.1 提取核心信息

阅读论文后，提取以下关键信息：

```
研究对象：[物种/材料/系统]
研究因素：[自变量，如饲料配方、温度、药物浓度]
处理组数：[N组，包括对照]
实验周期：[时长]
检测层次：
  - 整体层：[生长、存活、表型指标]
  - 组织层：[器官、组织学、酶活性]
  - 分子层：[基因表达、蛋白、代谢物]
主要发现：[1-3个核心结论]
最优条件：[哪个处理组效果最好]
```

### 1.2 识别论文类型

| 论文类型 | 特征 | 推荐配图 |
|---------|------|----------|
| 因子试验 | 2+因素交叉设计 | 因子矩阵图、交互作用图 |
| 剂量效应 | 单因素梯度 | 剂量响应曲线、最优点标注 |
| 机制研究 | 通路验证 | 分子通路图、信号级联图 |
| 比较研究 | 品种/品系对比 | 并列比较图、差异热图 |
| 时间序列 | 动态变化 | 时间线图、阶段划分图 |

## 第二步：配图类型选择

### 2.1 必备配图 vs 可选配图

**几乎所有论文都需要：**
- [ ] 实验设计示意图 (Fig. 1)
- [ ] 图文摘要 (Graphical Abstract)

**根据内容选择：**
- [ ] 代谢/信号通路图（有基因表达数据时）
- [ ] 组织器官结果图（有组织学/酶活数据时）
- [ ] 机制总结图（有明确调控机制时）

### 2.2 配图数量建议

| 期刊类型 | 主图数量 | 补充图数量 |
|---------|---------|-----------|
| Nature/Science/Cell | 4-6 | 8-15 |
| 专业顶刊 | 6-8 | 5-10 |
| 普通SCI | 5-8 | 3-5 |
| 学位论文 | 不限 | 不限 |

### 2.3 配图类型决策树

```
论文有基因表达数据?
├── 是 → 需要代谢通路图
│   └── 涉及多个通路? → 需要通路交互图
└── 否 → 跳过

论文有组织学/酶活数据?
├── 是 → 需要组织器官结果图
└── 否 → 跳过

论文有明确的调控机制?
├── 是 → 需要机制总结图
└── 否 → 用图文摘要替代

论文是因子设计?
├── 是 → 需要因子矩阵实验设计图
└── 否 → 使用流程式实验设计图
```

## 第三步：配图内容规划

### 3.1 实验设计图内容清单

```
必须包含：
- [ ] 实验动物/材料图示
- [ ] 处理组分组方式
- [ ] 样本量和重复数
- [ ] 实验周期
- [ ] 采样时间点
- [ ] 检测指标分类

可选包含：
- [ ] 环境条件（温度、海拔等）
- [ ] 饲料/处理组成比例
- [ ] 最优组标注
```

### 3.2 图文摘要内容清单

```
必须包含：
- [ ] 研究对象视觉呈现
- [ ] 主要处理因素
- [ ] 核心发现（1-3个）
- [ ] 最优条件标注

可选包含：
- [ ] 多层级响应（分子→组织→整体）
- [ ] 调控通路简图
- [ ] 关键数据亮点
```

### 3.3 代谢通路图内容清单

```
必须包含：
- [ ] 相关代谢通路（糖/脂/蛋白/免疫等）
- [ ] 关键基因/酶名称
- [ ] 调控方向（↑↓）标注
- [ ] 处理组间差异

可选包含：
- [ ] 分子结构图示
- [ ] 细胞/亚细胞定位
- [ ] 通路间交互
- [ ] 定量数据（柱状图/热图）
```

## 示例：虹鳟饲料论文分析

### 提取的核心信息

```
研究对象：三倍体虹鳟 (Oncorhynchus mykiss)
研究因素：饲料碳水化合物-脂质比例 (C:L)
处理组数：3组 (C:L=0.8, 1.0, 1.6)
实验周期：84天
检测层次：
  - 整体层：生长性能、血液生化
  - 组织层：肝脏、肠道、肌肉酶活性
  - 分子层：糖代谢、脂代谢、免疫基因
主要发现：C:L=1.0最优，平衡代谢与生长
最优条件：Diet 2 (C:L=1.0)
```

### 配图规划

| 图号 | 类型 | 目的 | 包含内容 |
|------|------|------|----------|
| Fig. 1 | 实验设计 | 展示试验布局 | 鱼、3种饲料、网箱、采样流程 |
| GA | 图文摘要 | 论文核心概要 | 鱼、三层响应、关键发现 |
| Fig. 2 | 代谢通路 | 展示分子机制 | 糖/脂/免疫通路、基因调控 |
| Fig. 3 | 组织结果 | 展示器官响应 | 肝脏/肠道/肌肉面板 |
